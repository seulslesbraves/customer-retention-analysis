# Customer Retention Analysis

## Project Overview
Analysis of customer churn patterns using the IBM Telco Customer Churn dataset.
The goal is to identify key factors that drive customers to leave,
and suggest strategies to improve retention.

## Dataset
- Source: IBM Telco Customer Churn Dataset
- 7,043 customers, 21 columns
- Target variable: Churn (Yes/No)

## Tools Used
- Python, Pandas, Matplotlib, Seaborn
- Jupyter Notebook

## Key Findings
1. Month-to-month customers churn 14x more than two-year contract customers
2. Nearly 1 in 2 new customers leave within their first year
3. Churned customers paid ~$15 more per month than retained customers

## How to Run
1. Clone this repository
2. Open `notebook.ipynb` in Jupyter Notebook
3. Run all cells from top to bottom
